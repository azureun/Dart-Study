### dart 특징

> null safety
- 프로그램 안전하게 해줌.

### flutter가 dart를 택한 이유
1. dart는 JIT와 AOT 2개 다 존재.
    1. 바로 결과 확인 가능.
2. 구글이 만들었음.
    1. react는 js 수정이 불가능.
    2. dart는 언어 수정해서 flutter를 빠르게 만들 수 있음.
